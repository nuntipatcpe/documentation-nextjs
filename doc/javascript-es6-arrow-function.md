# Arrow Function
- Original Function
    <code language="jsx">
    function originalName(p,p1){
        return p+p1;
    }</code>

- Arrow Function **Recommend**
    <code language="jsx">
    originalName = (p,p1)=>{
        return p+p1; 
    }</code>

    or
    <code language="jsx">
    originalName = (p,p1) => p+p1 
    </code>
