# Export

- **export**

  ```
  export let user = {
    id: "boom",
    password: "1234",
  };
  ```

- Useage **{ user }**

  ```
  import { user } from './path';
  ```

---

- **export Default**

  ```
  import React from 'react'
  function App() {
    return (
      <div>app</div>
    )
  }
  export default App;
  ```

- Usage **App**

  ```
  import App from './path';
  ```
