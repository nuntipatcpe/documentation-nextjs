# Install

With NPM

```
npx create-next-app@latest
```

With yarn

```
yarn create next-app
```

With pnpm

```
pnpm create next-app
```

# TypeScript

**New Project**

With NPM

```
npx create-next-app@latest --ts
```

With yarn

```
yarn create next-app --typescript
```

With pnpm

```
pnpm create next-app --ts
```


**To get started in an existing project**

```
touch tsconfig.json
```
