# Scrollbar custom
- scroll bar styles

```
::-webkit-scrollbar {
    width: 0.8rem;
    height: 0.8rem;
}
```
- Background bar

```
::-webkit-scrollbar-track {
      background: #f1f1f1;
}
```
- Bar

```
::-webkit-scrollbar-thumb {
      background: #888;
}
```
- Hover

```
::-webkit-scrollbar-thumb {
      background: #888;
}
```
